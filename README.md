# Menu do laboratório e do suporte técnico

Este projeto em Python foi cirado para praticar estrutura **match** e **case**.

O programa solicita a inserção de uma das opções listadas para o uso do laboratório, retornando com informações sobre o item escolhido. Também retorna ao status do pedido de manutenção técnica.
